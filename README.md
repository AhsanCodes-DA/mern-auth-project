# MERN Authentication App

A full-stack MERN Authentication System* built using *MongoDB, Express, React, and Node.js*, featuring secure login, 
registration, JWT authentication, OTP-based password reset, and real-time notifications.


##  Overview

This project provides a complete authentication flow with:-
- Secure JWT authentication stored in cookies  
- Email OTP for password reset (via Nodemailer & Gmail SMTP)
- Tailwind-powered responsive UI  
- Notification system for user actions (success/error)

Ideal for production-level MERN projects or as a starter template for secure web apps.
